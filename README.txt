#Data Scraping

-If you have not downloaded anaconda, please download with instructions: https://docs.anaconda.com/free/anaconda/install/windows/

###Create a new Conda virtual environment named "Scraping" with Python version 3.9. 
$conda create -n Scraping python=3.9
###Activate the virtual environment named "Scraping."
$conda activate scraping
###Upgrade the Pip package manager to the latest version.
$pip install --upgrade pip
###Install the libraries and dependencies listed in the "requirements.txt" file.
$pip install -r requirements.txt
